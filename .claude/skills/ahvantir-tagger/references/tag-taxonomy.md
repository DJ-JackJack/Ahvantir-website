# Ahvantir Tag Taxonomy

Tags are lowercase-with-hyphens. Apply a tag when the article meaningfully
discusses the concept — not just a passing mention.

## Content / meta tags

| Tag | Apply when… |
|---|---|
| `dm-only` | Content is secret/spoiler; has `[!warning]` callouts or `dm_only: true` |
| `homebrew` | Article describes a custom mechanic, item, or rule (not pure lore) |
| `stub` | Article is clearly incomplete — just a few placeholder sentences |
| `geography` | Physical geography, terrain, maps are the focus |
| `law-politics` | Laws, governance, political intrigue |
| `trade-economy` | Commerce, trade routes, currency |
| `military` | Warfare, tactics, military history |
| `religion-ritual` | Religious ceremonies, rituals, sacred sites |
| `natural-history` | Flora, fauna, ecology of Ahvantir |
| `magic-material` | Specific magical materials or substances |

## Guidelines for inferring new tags

When an article contains a proper noun not covered above:
- Named locations → add as `<location-name>` (e.g. `temple-of-the-silent-path`)
- Named NPCs → add as `<first-last>` or `<title-name>` (e.g. `elder-vayne`)
- Named events → add as `<event-name>` (e.g. `siege-of-stonegate`)
- Named organisations → add as `<org-name>` (e.g. `silver-thread-guild`)

Keep new tags lowercase-with-hyphens and as short as unambiguous.
